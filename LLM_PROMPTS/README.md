# LLM Prompts

Reusable prompts for starting new AI/dev sessions in this monorepo.

## Files

- `SESSION_BOOTSTRAP.md` — full context-building prompt (recommended default)
- `SESSION_BOOTSTRAP_SHORT.md` — compact daily/standup version

## How to use

1. Open a new AI/dev session.
2. Paste one of the bootstrap prompts as your first message.
3. Let the assistant complete context gathering before implementation.
