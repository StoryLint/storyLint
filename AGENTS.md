# AI/Dev Assistant Guide

Start here before beginning work in this repository.

## Primary entrypoint

- Read `LLM_PROMPTS/README.md` for available prompt templates.
- Use `LLM_PROMPTS/SESSION_BOOTSTRAP.md` at the top of a new session.

## Working rule

- Do a read-only context pass first.
- Do not implement changes until explicitly asked.
- Summarize current state, runbook, and next 3 prioritized tasks.
